Welcome to Corey's senior project repository

Follow the steps below to use WebLearn:

  1. Install node.js through the installer at https://nodejs.org/en/download/prebuilt-installer
  2. Clone the github repository/download the zip file and navigate to the WebLearn folder in a terminal/vscode terminal. https://github.com/StetsonMathCS/project-repository-coreysipe
  3. After navigating to the WebLearn run the command "npm i"
  4. Once completed run the command "npm run dev"
  5. After this you will see it hosted the application on http://localhost:3000 . Go to this link and explore WebLearn!

If you are experiencing any issues utilizing this application please contact csipe@stetson.edu
